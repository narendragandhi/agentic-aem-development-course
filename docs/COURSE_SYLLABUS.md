# Course Syllabus

## Agentic AEM Development Course

---

## Course Overview

**Title:** Agentic Development for AEM - BMAD, BEAD & GasTown  
**Subtitle:** Build production-ready AEM workflows with AI-driven development methodologies  
**Level:** Intermediate to Advanced  
**Duration:** 25 hours of video content  
**Prerequisites:** Java basics, AEM fundamentals

---

## Learning Outcomes

After completing this course, you will be able to:

1. **Apply BMAD methodology** to plan and execute AEM projects
2. **Use BEAD** for AI-friendly task tracking
3. **Implement GasTown** for multi-agent orchestration
4. **Build TDD workflows** with JUnit 5
5. **Create secure asset workflows** with antivirus scanning
6. **Implement security scanning** with OWASP patterns
7. **Configure AEM Cloud** deployments
8. **Optimize AEM performance** with Oak indexing

---

## FAQ

**Q: Do I need an AEM license?**  
A: No, the prototype uses mock services. For full AEM development, you'll need access to AEM SDK or AEM Cloud.

**Q: How long does it take?**  
A: At 2 hours/week = 12.5 weeks (3 months). At 5 hours/week = 5 weeks.

**Q: Is this for beginners?**  
A: Intermediate. You should know Java and have basic AEM knowledge.

**Q: Can I use this for work?**  
A: Yes, the code is MIT licensed.
